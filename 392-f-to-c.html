2. 392 f berapakah kelvin


SUHU dan KALOR
- Skala termometer

392 ºF = ___ K

secara bertahap


tF → tC

tC = ⁵/₉ (tF - 32)
tC = ⁵/₉ (392 - 32)
tC = ⁵/₉ · 360
tC = 200 ºC


tC → tK

tK = tC + 273
tK = 200 + 273
tK = 473 K  ← jwb



atau dihitung secara langsung
tK = ⁵/₉ (tF - 32)    + 273
tK = ⁵/₉ (392 - 32)  + 273
tK = ⁵/₉ · 360         + 273
tK = 200                + 273
tK = 473 K
392°F = (392 - 32) x 5/9 = 200°C

200°C = 200 + 273 = 473 K
