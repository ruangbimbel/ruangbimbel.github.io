PEMBAHASAN

Perlu di ingat bahwa dalam menerjemahkan suatu kalimat, tidak semua kata harus diterjemahkan. Kenapa? Karena ada beberapa kata yang berfungsi hanya sebagai kata bantu.

Berikut ini adalah kosa kata dari terjemahan dialog di atas :

Cute = Lucu/imut/menggemaskanWhat is its name = Apa namanya? (Untuk menanyakan nama suatu benda) Female = BetinaFur = BuluAround = SekitarEyes = MataThink so = Berpikir begituAnyway = Bagaimanapun/ngomong-ngomongFeed = Memberi makanRice = NasiSalty fish = Ikan asinMeat = DagingNo wonder = Pantas/tidak heranLooks = TerlihatFat = GemukHealthy = Sehat
