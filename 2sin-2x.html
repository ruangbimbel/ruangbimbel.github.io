14. Himpunan penyelesaian 2sin (2x+3π/2) = √3​


Himpunan penyelesaian dari 2 sin (2x + 3π/2) = √3 adalah {x| x = -7π/12 + k. π, x = -5π/12 + k. π, k bilangan bulat}.

Penjelasan dengan Langkah-Langkah

Diketahui:

2 sin (2x + 3π/2) = √3

Ditanyakan:

Himpunan Penyelesaian

Jawab:

2 sin (2x + 3π/2) = √3

sin (2x + 3π/2) = √3/2

sin (2x + 3π/2) = sin (π/3)

Maka:

1. 2x + 3π/2 = π/3 + k. 2π

2x = π/3 - 3π/2 + k. 2π

2x = -7π/6 + k. 2π

x = -7π/12 + k. π

2. 2x + 3π/2 = (π - π/3) + k. 2π

2x + 3π/2 = 2π/3 + k. 2π

2x = 2π/3 - 3π/2 + k. 2π

2x = -5π/6 + k. 2π

x = -5π/12 + k. π

Jadi, himpunan penyelesaiannya adalah {x| x = -7π/12 + k. π, x = -5π/12 + k. π, k bilangan bulat}.

18. Fungsi Trigonometri Yang Tepat Untuk Grafik Diatas AdalahA.)Y=2Sin XB.)Y=2Sin 2XC.)Y=2Sin 3XD.)Y=2Cos 2XE.)Y=2Cos3XTolong Bgt Ga Ngerti​


Memakai fungsi sinus karena titik awal berada di (0,0).

Jika cosinus seharusnya titik awal berada di (0,2)

Rumus umum fungsi sinus trigonometri adalah:

y = A Sin b(a ± x) ± c

dengan ketentuan:

A = Amplitudo

b = banyak gelombang dalam satu periode

a = pegeseran grafik secara horizontal

c = pergeseran grafik secara vertikal

Cara mencari:

1. A

yaitu dengan rumus: (titik puncak atas - titik puncak bawah)/2

Pada soal titik puncak atas = 2

titik puncak bawah = -2

Maka A = (2 - (-2)/2

= 4/2 = 2

Lalu nilainya (+) karena grafik bergerak ke atas terlebih dahulu.

2. b

yaitu dengan rumus 360/lamda. Dimana lamda adalah panjang satu gelombang. Pada soal, satu gelombang yaitu:

setengah gelombang = 90°

satu gelombang = 180°

(Itu soalnya salah. Seharusnya tanda 180° berada di sampinya 90°. Setelah ada satu bukit itu kan ada tanda 90°. Nah seharusnya setelah melewati satu lembah, ada tanda 180°)

Maka b = 360/180 = 2

3. a

Karena pada soal titik awal berada di sumbu x = 0 maka a = 0

(tidak memiliki pergeseran secara horizontal)

4. c

Karena titik awalnya berada di sumbu y = 0 maka c = 0

(tidak memiliki pergeseran secara vertikal)

Maka fungsi trigonometri yang tepat adalah:

y = 2Sin 2(0 + x) + 0

⇒ y = 2Sin 2x
