29. Biaya minimun dari 2x3 -2.100x2+600.000x adalah


B(x) = 2x³ - 2.100x² + 600.000x

B'(x) = 6x² - 4.200x + 600.000
     0 = 6x² - 4.200x + 600.000
     0 = x² - 700 + 100.000
            (x-200) (x-500)
Jadi, x₁ = 200 atau x₂ = 500

Masukkan x₁ dan x₂ ke B(X) satu per satu
Untuk x₁ B(200) = 2(200)³ - 2100(200)² + 600.000(200)
                        = 2(8000000) - 2100(40000) + 120000000
                        = 16000000 - 84000000 + 120000000
                        = 52000000

Untuk x₂ B(500) = 2(500)³ - 2100(500)² + 600.000(500)
                        = 2(125000000) - 2100(250000) + 300000000
                        = 250000000 - 525000000 + 300000000
                        = 25000000

Jadi biaya maksimumnya 52.000.000 rupiah
Jadi biaya minimumnya 25.000.000 rupiah (karena hasil x₂<x₁)
