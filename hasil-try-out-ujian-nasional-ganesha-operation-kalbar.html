7. Grafik hasil try out matematika adalah sebagai berikut ! Tentukan nilai rata-rata dari hasil try out !​


Statistika

Mean = jumlah data / banyak data

Median = nilai atau data tengah setelah diurutkan

Modus = nilai atau data yang sering muncul

Pembahasan

Diketahui

Nilai 20 = 1 murid

Nilai 40 = 6 murid

Nilai 60 = 7 murid

Nilai 80 = 9 murid

Nilai 100 = 2 murid

Mean

Mean = jumlah data / banyak data

Mean = [(20 x 1) + (40 x 6) + (60 x 7) + (80 x 9) + (100 x 2) / 25]

Mean = [(20) + (240) + (420) + (720) + (200) / 25]

Mean = [1600 / 25]

Mean = 64
