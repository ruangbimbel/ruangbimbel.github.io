Jawaban:

Jadi jumlah soalnya ada 40 buah

Penjelasan dengan langkah-langkah:

Diketahui

skor benar = 2

skor salah = -1

skor tidak menjawab = 0

Ririn menjawab

benar = 29

tidak menjawab= 5

Skor Ririn = 52

Ditanya banyak Soal = ?

Jawab

Skor = benar + salah + tidak menjawab

52 = ( 29 x 2) + salah + (5 x 0)

52 = 58 + salah + 0

52 = 58 + salah

salah = 58 - 52

salah = 6

Berarti jumlah soalnya

= benar + salah + tidak menjawab

= 29 + 6 + 5

= 40 soal

Jadi jumlah soalnya ada 40 buah

Uji

52 = (29 x 2) + ( 6 x -1 ) + ( 5 x 0 )

52 = 58 + ( -6) + 0

52 = 58 - 6

52 = 52 ( terbukti)
