9. Kerucut

Dik:

Keliling: k = 31,4 cm

Garis Pelukis: s = 13 cm

Dit:

Tinggi Kerucut?

untuk menghitung nya, kita cari dulu panjang jari-jari nya:

r = K / 2 π

r = 31,4 / 2 x 3,14

r = 31,4 / 6,28

r = c cm

Maka kita masukan ke rumus dengan cara Teorema Phytagoras:

t = √s² - r²

t = √13² - 5²

t = √169 - 25

t = √144

t = 12 cm

10. Luas Permukaan Setengah Bola

Dik:

Diameter Bola: d = 140 meter

maka, r = ½ x d = ½ x 140 = 70 meter

Harga Pembuatan = 308.000.000

Dit:

Harga Permeter?

Jawab:

Untuk mencari harga boaya permeter nya kita harus mencari luas permukaan setengah bola:

L = 4 π r²

karena bola berbentuk setengah lingkaran maka:

L = ½ x 4 x π x r²

L = ½ x 4 x 22/7 x 70²

L = 30.800 m²

Harga permeter = Hatga Total / Luas Permukaan

Harga Per Meter = 308.000.000 / 30.800

Harga Per Meter = 10.000

Apersepsi Materi :A. Bilangan Bulat

Apakah itu Bilangan bulat? Bilangan bulat adalah bilangan yang berbentuk positif, nol atau negatif bukan berbentuk desimal dan bentuk lainnya. contoh bilangan bulat adalah { ... -2 , -1 , 0 , 1 , 2 ... }

B. Operasi Bilangan Bulat

Operasi bilangan bulat secara umum ada penjumlahan, pengurangan, perkalian, dan pembagian.

1) Operasi Penjumlahan

Secara umum operasi penjumlahan adalah sebagai berikut :

a) Jika kedua angka sama tandanya ( + dengan + ),( - dengan - ) maka operasi nya bertambah dengan tanda yang sama contohnya :

2 + 1 = 3

- 2 - 1 = -3

b) Jika kedua angka berbeda tandanya, maka kurangi bilangan angka yang lebih besar dikurangkan dengan angka yang lebih kecil kemudian hasilnya menggunakan tanda dengan nilai yang paling besar contohnya :

- 2 + 1 = -1

2 + ( -1 ) = 1

2) Operasi Pengurangan

Secara umum operasi pengurangan adalah sebagai berikut :

a) Jika kedua angka sama tandanya (+ dengan +) contohnya :

2 - 1 = 11 - 2 = -12 - 2 = 0

b) Jika kedua angka sama tandanya ( - dengan - ) maka ( jika tanda (-) ketemu pengurangan maka tandanya menjadi +) contohnya :

(-2) - (-1) = (-2) + 1 = -1(-1) - (-2) = (-1) + 2 = 1

c) Jika angka berbeda tandanya (+ dengan - atau sebaliknya) maka ( jika tanda (-) ketemu pengurangan maka tandanya menjadi +) contohnya :

(-2) - (1) = -3(2) - (-1) = 2 + 1 = 3
