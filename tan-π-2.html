11. Nilai dari lim x→π 2 ((π − 2x) · tan 5x) =​


Untuk menyelesaikan soal ini kita akan menggunakan aturan batas dalam kalkulus. Berikut adalah langkah-langkahnya:

Pertama kita substitusikan x dengan π/2 - t. Dengan substitusi ini saat x mendekati π/2 dari kiri t akan mendekati 0.

Jadi kita memiliki:

lim (x → π/2) ((π - 2x) * tan(5x))

= lim (t → 0) ((π - 2(π/2 - t)) * tan(5(π/2 - t)))

Sederhanakan persamaan di atas:

= lim (t → 0) ((π - π + 2t) * tan(5(π/2 - t)))

= lim (t → 0) (2t * tan(5(π/2 - t)))

Berikutnya kita menggunakan aturan batas trigonometri yang menyatakan lim (θ → 0) tan(θ) = θ:

= lim (t → 0) (2t * 5(π/2 - t))

= lim (t → 0) (10t(π/2 - t))

Selanjutnya kita bisa mengalikan dan menyederhanakan persamaan di atas:

= lim (t → 0) (10t * (π/2) - 10t^2)

= lim (t → 0) (10t * π/2 - 10t^2)

= lim (t → 0) (5πt - 10t^2)

= 0 - 0

= 0

Jadi nilai dari lim x→π/2 ((π - 2x) * tan(5x)) adalah 0.
