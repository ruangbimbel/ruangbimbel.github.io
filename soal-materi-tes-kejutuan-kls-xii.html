Σε + ΣI.R = 0

- ε3 + ε1 + I1(R1 + r1) + I3(R3 + r3) = 0

3I1 + 6I3 = - 4 .... #1

~> Tinjau loop 2!

Σε + ΣI.R = 0

- ε2 - ε3 + I2(R2 + r2) + I3(R3 + r3) = 0

3I2 + 6I3 = 8 .... #2

~> I3 = I1 + I2

I2 = I3 - I1 .... #3

~> Subtitusi pers #3 ke #2, maka akan didapatkan :

3(I3 - I1) + 6I3 = 8

9I3 - 3I1 = 8 .... #4

~> Eliminasi pers #4 dgn #1, maka akan didapatkan :

15I3 = 4

I3 = 4/15

= 0,266 A ✓✓

3I1 = - 4 - 6(0,266)

= - 4 - 1,6

I1 = 5,6/3

29. Pertanyaan FisikaMateri : Listrik DinamisKelas : XII SMASoal : Pada gambar




GGL total
ΣE = E1 + E2 + E3
ΣE = -8 + 6 + 8 = 6 V

Hambatan dalam total
Σr = 0,5 + 0,5 + 1 = 2 ohm

Hambatan luar total
ΣR = 2 + 2 + 3 + 4 + 4 + 8
ΣR = 23 ohm

Kuat arus
i = ΣE / (ΣR + Σr)
i = 6 / (23 + 2)
i = 0,24 A
searah putaran jarum jam

Energi listrik pada hambatan luar
W = i² ΣR t
W = 0,24² • 23 • 30
W = 39,744 J

Energi listrik pada hambatan dalam (hilang memjadi panas)
Wr = 0,24 • 2 • 30 = 3,456 J

Energi total
W = V i t
W = 6 • 0,24 • 30 = 43,2 J
atau
W = 39,744 + 3,456 = 43,2 J
