30. bedanya has/had , has been / had been apa?


Has+been merupakan auxiliary verb yang digunakan untuk membentuk: 

1. Present perfect continuous tense 
Rumus: S + has/have been + present participle 
Contoh: She has been driving for an hour. 

2. Passive present perfect tense 
Rumus: S + has/have been + past participle 
Contoh: The task has been done. 

Sedangkan 

Had+been merupakan auxiliary verb yang digunakan untuk membentuk: 

1. Past perfect continuous tense 
Rumus: S + had been + present participle 
Contoh: You had been soaking your clothes all day. 

2. Passive past perfect tense 
Rumus: S + had been + past participle 
Contoh: The room had been painted in white.

had-------> untuk past

33. if everyone takes care of the environment well, our world....a.would be much betterb. has been much betterc.will be much betterd.is much bettere.is being much better​


║CONDITIONAL SENTENCE║

Answer

C. will be much better

Discussion

"If everyone takes care of the environment well, our world will be much better." { Jika setiap orang mempedulikan lingkungan dengan baik, bumi kita akan jauh lebih baik }

Kalimat diatas merupakan kalimat bersyarat/ Conditional sentence , dimana terdapat -if clause dan result clause.

Untuk menentukan kata kerja pada result clause, harap cermati bentuk kata kerja pada -if clause nya yaitu "takes care of" dan konteks kalimatnya.  Ini merupakan petunjuk utama terkait dengan tipe conditional sentence yang digunakan dalam kalimat tersebut.

"Takes care of" merupakan phrasal verb bentuk simple present tense. Maka, ini termasuk kalimat Conditional tipe ke 1. Jika kita pahami konteks kalimatnya, merujuk pada kemungkinan yang akan tiimbul bahwa bumi kita akan lebih baik jika setiap orang peduli lingkungan. Jadi jawaban yang tepat adalah "will be much better" karena sampai sekarang belum terbukti kebenarannya dan kemungkinan akan terjadi jika persyaratan utama nya terpenuhi dahulu yaitu "takes care of the environment."

Explanation

CONDITIONAL SENTENCE / Andai-andai

Conditional sentence tells about the condition that will be actualized ( possible condition ) or only dreaming ( intention ) because of the particular condition.

Conditional sentence structure consists -If clause and result clause ( main clause ).

Use Comma when the result clause (main clause ) is started at the beginning sentence. Otherwise, there is no comma when -IF clause is put after the result clause.

There are 4 types of Conditional Sentence

1. ZERO CONDITIONAL

The Zero Conditional is used to describe the PHENOMENON,  and GENERAL TRUTH ( THE FACT  )

Zero Conditional uses the simple present tense form to reflect the truth.

Pattern :

IF+S+base verb (s/es ) + O/C , S+base verb (s/es ) + O/C

Example :

If we boil water , it evaporates up into atmosphere.

{ Jika kita merebus air, itu akan menguap ke udara }

Analysis :

It tells about the general truth or phenomenon.

It uses the simple present tense form to describe the truth. It is proven that the water evaporates up and disappear when it is heated at 100 degrees Celsius.

2. TYPE 1 CONDITIONAL

The Type 1 Conditional is used to describe the truth  in the present/future ( The planning (prediction)  will be probably happened )

In fact, the speaker will actualize his/her dream now or soon .

Pattern :

If +S+ verb (s/es) + O, S+will+verb1+O/C

Example :

If I study hard, I will pass the exam easily.

Analysis :

Meaning : it's possible that I will be successful for the exam later.

It uses simple future form in its main clause / result clause to describe the possible condition and state the prediction will be true.

3. TYPE 2 CONDITIONAL

The Type 2 Conditional type  is used to describe  the untruth in the present / future

It is Contrary to the fact in the present or it could be in the future

The speaker has an intention ( dream ) but he/she can't afford his/her dream comes true in the present/future

"Keep in mind, if we use the first , second person and third person singular  ( I, she, he, it , ) in the nominal sentence , it must be followed by the PAST linking verb "were."" It is the same way to the plural form as the formal writing." If we use the PAST linking verb "was " , it should be used in the colloquial as the informal conversation.

Pattern:

If+S+verb2+O/C , S+could/could/might+verb1+O/C

Example :

If I were a president, I would build the biggest library in Jakarta.

Analysis :

Meaning :  I can't build the biggest library because I'm not a president now.

It uses the simple past form to reflect the untruth condition in the present  and the speaker's intention will not come true because it is contrary to the real condition for now or maybe future.

4. TYPE 3 CONDITIONAL

The Type 3 Conditional  Is used to describe  the untruth in the past

It is contrary to the fact in the past and It  reflects a regret for something that the speaker couldn't afford to do in the past.

Pattern :

If+S+Had+Verb 3+O/C  , S+Would/could/might+have+Verb 3+O/C

Example :

If I had known my friend's house address correctly, I wouldn't have been lost.  

Analysis :

Meaning :  I didn't know my friend's house address so I was lost.
