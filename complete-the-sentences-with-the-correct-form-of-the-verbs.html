Jawaban :StartedLedEncouragedBeganSetPEMBAHASANSIMPLE PAST

➡ Tenses yang menjelaskan tentang suatu kegiatan ataupun kejadian yang terjadi pada masa lampau.

RUMUS :

(+) S + V2 + O

They opened the door. She swept the floor.

(-) S + Did + not + V1 + O

They did not open the door. She did not sweep the floor.

(?) Did + S + V1 + O

Did they open the door? Did she sweep the floor?

Pada soal diatas kita diperintahkan untuk mengubah kata kerjanya menggunakan regular/irregular verbs. Karena kalimatnya bentuk lampau dan kalimatnya bentuk positif semua, maka kata kerjanya harus di ubah ke bentuk verb-2, dan berikut adalah perubahannya :

Regular Verbs

➡ Kata kerja beraturan, dimana bentuk verb-2 dan ketiganya tidak jauh berbeda dari kata aslinya (verb-1) karena hanya tinggal menambahkan -d/ed.

Start => Started => StartedEncourage => Encouraged => Encouraged

Irregular Verbs

➡ Kata kerja tidak beraturan, dimana bentuk verb-2 dan verb-3 nya sudah berbeda dari kata aslinya (verb-1)

Lead => Led => LedBegin => Began => BegunSet => Set => Set
