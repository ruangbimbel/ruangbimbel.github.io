7. Sri utami ,S.H.Seorang PNS golongan IV/a.Pada bulan maret memiliki gaji pokok 4.500.000.Tunjangan istri 10%,tunjangan anak 2% per anak,tunjangam fungsional 3.000.000,iuran pengsiun 4,75%.Status kawin dengan 2 anak .Jumlah bruto perbulan adalah


PENGHASILAN BRUTO

Penghasilan yang belum dikurangi oleh biaya-biaya lain seperti biaya jabatan, iuran pensiun, iuran asuransi, iuran jaminan kesehatan, dan lain-lain

________________________________

Langkah-Langkahnya

1. Mencari tunjangan istri

2. Mencari tunjangan 2 anak

3. Mencari tunjangan fungsional

4. Mencari jumlah bruto per bulan

GAJI POKOK

= 4.500.000

TUNJANGAN ISTRI

= 10% x gaji pokok per bulan

= 10% x 4.500.000

= 450.000

TUNJANGAN 2 ANAK

= 2 x (2% x gaji pokok per bulan)

= 2 x (2% x 4.500.000)

= 2 x 90.000

= 180.000

TUNJANGAN FUNGSIONAL

= 3.000.000

JUMLAH BRUTO PER BULAN

= gaji pokok + tunjangan istri + tunjangan 2 anak + tunjangan fungsional

= 4.500.000 + 450.000 + 180.000 + 3.000.000

= Rp8.130.000

Jadi, jumlah bruto per bulan PNS tersebut adalah Rp8.130.000

PENGHASILAN NETTO

Penghasilan bruto (gaji pokok + tunjangan-tunjangan) yang sudah dikurangi dengan biaya-biaya (biaya jabatan, iuran pensiun, iuran BPJS, iuran asuransi, dan lain-lain)

________________________________

Langkah-Langkahnya

1. Mencari gaji pokok

2. Mencari tunjangan istri

3. Mencari tunjangan 2 anak

4. Mencari tunjangan fungsional

5. Mencari iuran pensiun

6. Mencari biaya jabatan

7. Mencari penghasilan netto

GAJI POKOK

= 3.500.000

TUNJANGAN ISTRI

= 10% x gaji pokok

= 10% x 3.500.000

= 350.000

TUNJANGAN 2 ANAK

= 2 x (2% x gaji pokok)

= 2 x (2% x 3.500.000)

= 2 x 70.000

= 140.000

TUNJANGAN FUNGSIONAL

= 2.000.000

IURAN PENSIUN

= 4,75% x gaji pokok

= 4,75% x 3.500.000

= 166.250

BIAYA JABATAN

= 5% x gaji pokok

= 5% x 3.500.000

= 175.000

PENGHASILAN NETTO

= (gaji pokok + tunjangan istri + tunjangan 2 anak + tunjangan fungsional) - (iuran pensiun + biaya jabatan)

= (3.500.000 + 350.000 + 140.000 + 2.000.000) - (166.250 + 175.000)

= 5.990.000 - 341.250

= Rp5.648.750

Jadi, penghasilan netto Kartika Sari per bulannya adalah Rp5.648.000
