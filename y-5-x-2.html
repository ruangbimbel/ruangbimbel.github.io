1. ╔══════════╗ ----QUIZ---- ╚══════════╝ Contoh Soal " [tex]2(x+y)^{2} +7(x+y)+5\\=((x+y)+1)(2(x+y)+5)\\=(x+y+1)(2x+2y+5)[/tex] Difaktorkan! : a. 2(x+y)²+5(x+y)+3 =... b. 2(x+y)²+(x+y)-3 = ..... c. 2(x+y)²-9(x+y)-5 = ..... d. 3(x+y)²-13(x+y)+4 = .....


[tex]\boxed{\bold{a.}}[/tex]

2(x + y)² + 5(x + y) + 3

= ( 2(x + y) + 3 )( (x + y) + 1 )

= ( 2x + 2y + 3 )( x + y + 1 )

[tex]\boxed{\bold{b.}}[/tex]

2(x + y)² + (x + y) - 3

= ( 2(x + y) + 3 )( (x + y) - 1 )

= ( 2x + 2y + 3 )( x + y - 1 )

[tex]\boxed{\bold{c.}}[/tex]

2(x + y)² - 9(x + y) - 5

= ( 2(x + y) + 1 )( (x + y) - 5 )

= ( 2x + 2y + 1 )( x + y - 5 )

[tex]\boxed{\bold{d.}}[/tex]

3(x + y)² - 13(x + y) + 4

= ( 3(x + y) - 1 )( (x + y) - 4 )

= ( 3x + 3y - 1 )( x + y - 4 )

Jawab:

Penjelasan dengan langkah-langkah:

2(x + y)² + 5(x + y) + 3

⇒ ( 2x + 2y + 3 )( x + y + 1 )

⇒( 2(x + y) + 3 )( (x + y) - 1

⇒ ( 2x + 2y + 3 )( x + y - 1 )

2(x + y)² - 9(x + y) - 5

⇒ ( 2x + 2y + 1 )( x + y - 5 )

⇒ ( 3(x + y) - 1 )( (x + y) - 4 )

x + y = 5 (pers. 1)

x - y = -2 (pers. 2)

› Evaluasi pers. 2

x - y = -2

x = -2 + y

› Substitusi nilai x ke pers. 1

x + y = 5

-2 + y + y = 5

-2 + 1y + 1y = 5

-2 + 2y = 5

2y = 5 + 2

2y = 7

y = 7/2

› Substitusi nilai y ke nilai x

x = -2 + y

x = -2 + 7/2

x = -2/1 + 7/2

x = -4/2 + 7/2

x = 3/2

HP = {3/2 , 7/2}

KESIMPULAN:

∴ Jadi, himpunan penyelesaian dari x + y = 5 dan

x - y = -2 adalah {3/2 , 7/2}.
