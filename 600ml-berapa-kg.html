6. sebanyak 600mL larutan CH3CooH 0,1M dicampur dengan 600mL CH3CooNa 0,15M. (ka Ch3COOH = 1,8x10^-5). tentukan Ph campuran tersebut"


CH3COOH 0,1 M; vol = 600 ml
maka mol = M x V = 0,1 x 600 = 60 mmol
CH3COONa 0,15 M; vol = 600 ml
maka mol = M x V = 0,15 x 600 = 90 mmol
Ka = 1,8 x 10^-5

dilihat dari soalnya, larutan yang dihasilkan adalah buffer asam
sehingga pH dapat dicari menggunakan rumus :
[H+] = (Ka x mol asam)/mol garam
        = (1,8 x 10^-5 x 60)/90
        = 12.10^-5
pH   = - log [H+]
        = - log [12.10^5]
        = 5-log12

12. 400 cm kubik +600ml=....liter


Cm³, ml  dan liter merupakan satuan volume, cm³ satuan volume berbasis kubik sedangkan ml dan liter merupakan satuan volume berbasis liter.

Untuk satuan volume berbasis kubik dan liter dapat dilihat tangga dibawah ini ↓

km³

     hm³

           dam³

                 m³ (kilo liter)

                      dm³ (liter)

                            cm³ (mililiter)

                                  mm³

pada tangga ini jika turun 1 tangga kita kali 1000 dan sebaliknya jika naik 1 tangga kita bagi 1000

Pembahasan

Dari tangga diatas dapat kita ketahui bahwa m³ setara dengan kilo liter, dm³ setara dengan liter dan cm³ setara dengan mililiter

sehingga

400 cm³ + 600 ml = ... liter

400 ml + 600 ml = ... liter

dari ml dijadikan liter, lihat tangga diatas naik 1 tangga, berarti kita bagi 1000

400 ml + 600 ml = 1000 ml

1000 ml = 1000 : 1000 liter

             = 1 liter


Jadi 400 cm³ + 600 ml = 1 liter
