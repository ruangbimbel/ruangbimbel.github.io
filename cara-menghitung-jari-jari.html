17. cara menghitung panjang busur bila hanya diketahui 45° tanpa jari jarinya. cara mencari jari jari bila hanya yg diketahui sudutnya?​


Rumus mencari jari² lihat di gambar

Contoh soal

1. Sudut pusat suatu juring lingkaran adalah 45°, hitunglah jari-jari lingkaran jika luas juring adalah 77 cm². (π = 22/7)

Jawab:

Luas juring = 77 cm²

Sudut pusat = 45°

r = √[(luas juring x 360)/(sudut pusat x π)]

r = √[(77 x 360) / (45 x 22/7)

= √[(77 x 8) : 22/7

= √616 x 7/22

= √28 x 7

= √196

= 14 cm
