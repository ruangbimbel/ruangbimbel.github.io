PEMBAHASAN

Verb 2 Biasanya digunakan dalam Past Tense kalimat positive (+)

Contoh:

=> I bought a new bag yesterday

Structure Kalimatnya:

Subject => IVerb 2 => Bought Object => a bew bagAdverb of time => yesterday

tapi Past Tense juga menggunakan Verb 1 pada kalimat Negative (-) dan interogative (?)

Contoh:

=> I did not buy a new bag yesterday

Structure Kalimatnya:

Subject => Idid not => didn't Verb 1 => Buy Object => a bew bagAdverb of time => yesterday

=> Did I buy a new bag yesterday?

Structure Kalimatnya:

Auxilary => DidSubject => IVerb 1 => Bought Object => a bew bagAdverb of time => yesterday?

Pola yg digunakan:

(+) S + Verb 2 + O + adverb of time

(-) S + did not + Verb 1 + O + adverb of time

(?) did + S + Verb 1 + O + adverb of time?
