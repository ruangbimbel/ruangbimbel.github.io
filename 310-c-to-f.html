2. Pengukuran suhu ruangan menunjukkan 310 k maka skala °c, °f, dan °r, menunjukkan


310 k = 5/5 x (310 - 273)° C
= 1 x 37 = 37° c

310k = 9/5 x (310-273) + 32
= 9/5 x 37 + 32
= 98,6° f


310 k = 4/5 x (310 - 273)
= 4/5 x 37
= 29,6 ° rMata Pelajaran = Fisika
Materi = Konversi suhu C,F,R, K


Jika diketahui suhu pada kelvin, untuk mengubahnya ke satuan lain lebih mudahnya ubah dahulu ke celcius dengan menguranginya dengan 273 (ini sudah ketentuannya)

==Konversi Kelvin ke Celcius==

310 °K = 310-273 = 37 °C

Perbandingan Celcius : Reamur : Fahrenheit yaitu 5 : 4 : 9.
Tetapi, jika mengubah dari satuan lain ke satuan Fahrenheit, setelah mendapat hasil dari perbandingan, harus ditambah 32. Jika dari Fahrenheit ke satuan lain, sebelum dihitung menggunakan perbandingan, angkanya terlebih dahulu dikurang 32.

Untuk mempermudah dalam mengubah ke Fahrenheit, kita gunakan saja skala Celcius.

==Konversi Celcius ke Fahrenheit==

37 °C = 9/5 × 37 = 66,6 + 32 = 98,6 °F

Dalam mengubah ke Reamur, kita gunakan juga skala Celcius agar lebih mudah.

==Konversi Celcius ke Reamur==

37 °C = 4/5 × 37 = 29,6 °R

21. Pengukuran suhu ruangan menunjukkan 310 k maka skala Â°c, Â°f, dan Â°r, menunjukkan


0°C = 273K
jadi, °C = 310-273
= 37°C
untuk skala F gunakan perbandingan terhadap C yaitu, 9/5 x (skala dalam C) +32
9/5 x 37 +32
= 98.6°F
untuk R, yaitu dengan perbandingan dengan C, bisa juga dengan F, R:C = 4:5
4/5 x 37
= 29,6°RMapel : Fisika

Kata Kunci : konversi suhu

Pembahasan : 

Diketahui suhu yang diukur termometer Kelvin adalah 310°K. Pertanyaannya, berapa jika diukur dengan termometer Fahrenheit, Celcius dan Reamur.

Kelvin ke Celcius
(K - 273)/100 = C/100
K - 273 = C
310 - 273 = C
C = 37°C. ←

Kelvin ke Fahrenheit
(F - 32)/180 = (K - 273)/100
(F - 32)/180 = (310 - 273)/100
(F - 32)/180 = 37/100
F - 32 = 37/100 × 180
F - 32 = 66,6
F = 66,6 + 32
F = 98,6°F. ←

Kelvin ke Reamur
(K - 273)/100 = R/80
(310 - 273)/100 = R/80
37/100 = R/80
37/100 × 80 = R
29,6 = R
R = 29,6°R. ←

25. - 310°c =.... Reamur? - 150°f =.... Reamur? - 70 reamur =.... Fahrenheit? - 30°f =.... Kelvin? - 10kelvin =.... Reamur? NB: Dengan cara kerjanya yah Kawan.


Suhu adalah besaran yang menyatakan derajat panas dingin suatu benda. Alat yang digunakan untuk mengukur suhu adalah thermometer.


Pembahasan

Berdasarkan soal diatas diketahui bahwa:

310°C = ....°R? 150°F=.... °R? 70°R =.... °F? 30°F =.... °K? 10°K =....°R?

Adapun yang ditanyakan pada soal adalah suhu benda jika diukur pada skala reamur, fahrenheit dan kelvin.

Sebelum menyelesaikan soal diatas, ada bebrapa rumus untuk mengkonversi suhu dari skala satu ke skala lainnya adalah sebagai berikut :

1. Konversi suhu dari Celcius (C) ke Reamur (R), Fahrenheit (F), dan Kelvin (K) adalah:

R = (4/5) C

F = (9/5) C + 32

K = C + 273

2. Konversi suhu dari Reamur (R) ke Celcius (C), Fahrenheit (F), dan Kelvin (K) adalah:

C = (5/4) R

F = (9/4) R + 32

K = C + 273 = (5/4) R + 273

3. Konversi suhu dari Fahrenheit (F) ke Celcius (C), Reamur (R), dan Kelvin (K) adalah:

C = 5/9 (F-32)

R = 4/9 (F-32)

K = 5/9 (F-32) + 273

4. Konversi suhu dari Kelvin (K) ke Celcius (C), Reamur (R), Fahrenheit (F) adalah:

C = K – 273

R = 4/5 (K-273)

F = 9/5 (K-273) + 32

Nah, setelah diketahui beberapa rumus diatas, maka untuk pertama mencari suhu dalam skala reamur caranya sebagai berikut:

t°R = (4/5) C

t°R = 4/5 . 310

t°R = 1240/5

t°R = 248°R

Selanjutnya megubah suhu dari skala fahrenheit ke skala reamur caranya sebagai berikut:

t°R = 4/9 (F-32)

t°R = 4/9 (150 - 32)

t°R = 4/9 . 118

t°R = 472/9

t°R = 52,4°R

Selanjutnya megubah suhu dari skala reamur ke skala fahrenheit caranya sebagai berikut:

t°F = (9/4) R + 32

t°F = (9/4.70) + 32

t°F = (630/4) + 32

t°F = 157,5 + 32

t°F = 189,5°F

Selanjutnya megubah suhu dari skala fahrenheit ke skala kelvin caranya sebagai berikut:

t°K = 5/9 (F - 32) + 273

t°K = 5/9 (30 -32) + 273

t°K = (5/9. -2) + 273

t°K = -10/9 + 273

t°K = -1 + 273

t°K = 272°K

Selanjutnya megubah suhu dari skala kelvin ke skala reamur caranya sebagai berikut:

t°R = 4/5 (K-273)

t°R = 4/5 (10 - 273)

t°R = 4/5 . (-263)

t°R = -1052/5

t°R = -210,4°R

Dengan demikian dapat disimpulkan bahwa 310°C =248°R, 150°F= 52,4°R, 70°R = 189,5°F, 30°F = 272°K, 10°K = -210,4°R.
