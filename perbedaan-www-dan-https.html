1. apakah perbedaan www dan https?​


HTTPS atau hyper-text transfer protocol secure adalah serangkaian peraturan dalam komunikasi antara 2 system seperti komputer client dan web server. HTTPS menambahkan protocol secure untuk mengamankan transfer hyper-text 2 system tersebut dengan cara enkripsi dan dekripsi. Sedangkan WWW atau World Wide Web adalah prefix dari domain name system atau DNS untuk menyatakan bahwa website yang diakses menggunakan protocol HTTP/HTTPS dan dapat diakses melalui internet. WWW dapat digunakan sebagai penanda bahwa URL yang diakses adalah HTTP/HTTPS. Hal ini dapat membantu untuk membedakan file server yang menggunakan FTP dengan web server yang menggunakan HTTP/HTTPS sebagai transfer protocolnya.

19. Apa perbedaan antara WWW dan gopher


Kelas           : IX (3 SMP) 

Pelajaran     : TI 

Kategori      : Mengenal Internet

Kata Kunci  : Perbedaan, Gopher, WWW




Baik itu Gopher dan WWW adalah dua diantara sekian aplikasi internet yang dipergunakan untuk melakukan pencarian dan pendistribusian data berbasis internet. Meski begitu, terdapat perbedaan antara Gopher dan WWW, sebagai berikut:


❖  Gopher hanya pencarian berbasis teks saja sementara WWW mampu menelusuri teks dan media lainnya seperti gambar dan suara.

❖  Tampilan pada WWW lebih seperti brosur bergambar, sementara pada gopher tampilannya cenderung mirip daftar menu.

❖  Gopher mampu mencarikan data lebih cepat namun WWW memiliki kepampuan hypertext, hypermedia, tampilan grafik otomatis dan kemudahan penggunaan yang jauh lebih baik.

❖  WWW mampu terhubung dengan server gopher tetapi gopher tak bisa mengakses server www.


Gopher adalah pendahulu dari WWW namun popularitasnya semakin menurun sejak ada www. Meski begitu sampai saat ini gopher masih dipertahankan dan masih digunakan sebagai alternatif.

24. Tuliskan perbedaan dari www,website,web ?


# Web site (situs web) adalah keseluruhan isi web server yang dimiliki oleh seseorang atau lembaga tertentu. Web site bisa diibaratkan sebagai sebuah buku yang terdiri dari halaman-halaman tertentu, termasuk cover, daftar isi, dan sebagainya.
# World Wide Web (biasa disingkat WWW) adalah salah satu dari sekian banyak layanan yang ada di internet yang saling terkait dalam sebuah dokumen berformat hypertext yang berisi beragam informasi. Layanan ini paling banyak digunakan di internet untuk menyampaikan informasi karena sifatnya mendukung multimedia. Artinya informasi tidak hanya disampaikan melalui teks, tapi juga gambar, video dan suara. WWW dapat diakses melalui sebuah perangkat yang disebut web browser.
# Web pada awalnya merupakan ruang informasi dalam internet, dengan menggunakan teknologi hyperteks, terdiri dari link-link yang disediakan dalam dokumen web untuk menemukan informasi yang dibutuhkan oleh pengguna, ditampilkan melalui browser.

HTML (Hyper Text Markup Language) adalah sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web dan menampilkan berbagai informasi di dalam sebuah browser Internet. Bermula dari sebuah bahasa yang sebelumnya banyak digunakan di dunia penerbitan dan percetakan yang disebut dengan SGML (Standard Generalized Markup Language), HTML adalah sebuah standar yang digunakan secara luas untuk menampilkan halaman web. HTML saat ini merupakan standar Internet yang didefinisikan dan dikendalikan penggunaannya oleh World Wide Web Consortium (W3C).

HTML berupa kode-kode tag yang menginstruksikan browser untuk menghasilkan tampilan sesuai dengan yang diinginkan. Sebuah file yang merupakan file HTML dapat dibuka dengan menggunakan browser web seperti Mozilla Firefox atau Microsoft Internet Explorer. HTML juga dapat dikenali oleh aplikasi pembuka email ataupun dari PDA dan program lain yang memiliki kemampuan browser.


HTTP Singkatan dari Hypertext Transfer Protocol, yang mana adalah suatu protokol yang digunakan oleh World Wide Web. HTTP mendefinisikan bagaimana suatu pesan bisa diformat dan dikirimkan dari server ke client.

HTTP juga mengatur aksi-aksi apa saja yang harus dilakukan oleh web server dan juga web browser sebagai respon atas perintah-perintah yang ada pada protokol HTTP ini.
Sebagai contoh, ketika Anda mengetikkan suatu alamat atau URL pada internet browser Anda, maka sebenarnya web browser akan mengirimkan perintah HTTP ke web server.

Web server kemudian akan menerima perintah ini dan melakukan aktivitas sesuai dengan perintah yang diminta oleh web browser (misalnya akses ke database, file, e-mail dan lain sebagainya). Hasil aktivitas tadi akan dikirimkan kembali ke web browser untuk ditampilkan kepada pengguna.

39. jelaskan perbedaan antara WWW dan Gopher


Kelas           : IX (3 SMP) 
Pelajaran     : TI 
Kategori      : Mengenal Internet
Kata Kunci  : Perbedaan, Gopher, WWW



Baik itu Gopher dan WWW adalah dua diantara sekian aplikasi internet yang dipergunakan untuk melakukan pencarian dan pendistribusian data berbasis internet. Meski begitu, terdapat perbedaan antara Gopher dan WWW, sebagai berikut:

❖  Gopher hanya pencarian berbasis teks saja sementara WWW mampu menelusuri teks dan media lainnya seperti gambar dan suara.
❖  Tampilan pada WWW lebih seperti brosur bergambar, sementara pada gopher tampilannya cenderung mirip daftar menu.
❖  Gopher mampu mencarikan data lebih cepat namun WWW memiliki kepampuan hypertext, hypermedia, tampilan grafik otomatis dan kemudahan penggunaan yang jauh lebih baik.
❖  WWW mampu terhubung dengan server gopher tetapi gopher tak bisa mengakses server www.

Gopher adalah pendahulu dari WWW namun popularitasnya semakin menurun sejak ada www. Meski begitu sampai saat ini gopher masih dipertahankan dan masih digunakan sebagai alternatif.
