16. diketahui 3 Fe2O3 + CO menghasilkan 2 Fe3O4 + CO2 delta H= -48,5 Kj, Fe + CO2 menghasilkan FeO + CO delta H= -11,0 Kj, Fe3O4 + CO menghasilkan 3FeO + CO2 delta H= 22 Kj hitunglah delta H untuk reaksi; Fe2O3 + 3 CO menghasilkan 2 Fe + 3 CO2


Diketahui : 3Fe₂O₃ + CO ⇒ 2Fe₃O₄ + CO₂         ΔH = -48,5 kJ

                  Fe + CO₂ ⇒ FeO + CO                       ΔH = -11 kJ

                  Fe₃O₄ + CO ⇒ 3FeO + CO₂                ΔH = 22 kJ

Maka nilai ΔH untuk reaksi : Fe₂O₃ + 3CO ⇒ 2Fe + 3CO₂ adalah 20,5 kJ.

Penyelesaian Soal :

LANGKAH PERTAMA (I)

Berdasarkan reaksi target yang diinginkan, maka caranya yaitu :

3Fe₂O₃ + CO ⇒ 2Fe₃O₄ + CO₂         ΔH = -48,5 kJ

Fe + CO₂ ⇒ FeO + CO                      ΔH = -11 kJ  (Dibalik, Dikali 6)

Fe₃O₄ + CO ⇒ 3FeO + CO₂              ΔH = 22 kJ  (Dikali 2)

LANGKAH KEDUA (II)

Tahap 1 :

3Fe₂O₃ + CO ⇒ 2Fe₃O₄ + CO₂              ΔH = -48,5 kJ

2Fe₃O₄ + 2CO ⇒ 6FeO + 2CO₂            ΔH = 44 kJ

__________________________________________ +

3Fe₂O₃ + 3CO ⇒ 6FeO + 3CO₂            ΔH = -4,5 kJ

Tahap 2:

3Fe₂O₃ + 3CO ⇒ 6FeO + 3CO₂            ΔH = -4,5 kJ

6FeO + 6CO ⇒ 6Fe + 6CO₂                 ΔH = 66 kJ

_________________________________________ +

3Fe₂O₃ + 9CO ⇒ 6Fe + 9CO₂               ΔH = 61,5 kJ

LANGKAH KETIGA (III)

Karena pada reaksi target hanya diinginkan 1 koefisien Fe₂O₃, Kemudian dibagi dengan 3 sehingga menjadi :

3Fe₂O₃ + 9CO ⇒ 6Fe + 9CO₂               ΔH = 61,5 kJ     (Dibagi 3)

________________________________________

Fe₂O₃ + 3CO ⇒ 2Fe + 3CO₂               ΔH = 20,5 kJ

∴ Kesimpulan nilai ΔH adalah 20,5 kJ.

37. diketahui 3Fe2O3+CO 2 → Fe3O4 + CO2 ∆H= -48,5 kJ Fe + CO2 → FeO + CO ∆H= -11,0 kJ Fe3O4 + CO → FeO + CO2 ∆H=22 kJ Hitunglah ∆H untuk reaksi : Fe2O3 + 3CO → 2 Fe + 3 CO2


Kelas         : XI
Pelajaran   : Kimia
Kategori     : Termokimia
Kata Kunci : perubahan entalpi, ΔH, reaksi bertingkat

Ditanya
Perubahan entalpi (ΔH) pada reaksi: 
Fe₂O₃ + 3CO → 2Fe + 3CO₂
Selanjutnya kita sebut reaksi ini sebagai reaksi target

Pembahasan

Persoalan kali ini merupakan jenis persamaan termokimia tipe reaksi bertingkat untuk mencapai suatu reaksi target serta menghitung perubahan entalpi (nilai ΔH).

Sebelum perubahan 
Reaksi-1 ⇒ 3Fe₂O₃ + CO → 2Fe₃O₄ + CO₂     ΔH = -48,5 kJ
Reaksi-2 ⇒ Fe + CO₂ → FeO + CO              ΔH = -11,0 kJ 
Reaksi-3 ⇒ Fe₃O₄ + CO → 3FeO + CO₂          ∆H=22 kJ           

Setelah diamati dengan cermat pada ketiga reaksi di atas, perlakuan terhadap setiap reaksi untuk disesuaikan dengan reaksi target adalah sebagai berikut:
⇒ reaksi-1 dikalikan ¹/₃ (berikut ΔH), 
⇒ reaksi-2 dikalikan 2 (berikut ΔH) dan dibalik,
⇒ reaksi-3 dikalikan ²/₃ (berikut ΔH),
lalu ketiga persamaan termokimia langsung kita tambahkan ke bawah.

Sesudah perubahan 
Reaksi-1 ⇒ Fe₂O₃ + ¹/₃CO → ²/₃Fe₃O₄ + ¹/₃CO₂      ΔH = -16,17 kJ
Reaksi-2 ⇒ 2FeO + 2CO → 2Fe + 2CO₂               ΔH = +22,0 kJ 
Reaksi-3 ⇒ ²/₃Fe₃O₄ + ²/₃CO → 2FeO + ²/₃CO₂      ∆H = +14,67 kJ           
____________________________________________________(+)

Jawaban ⇒ Fe₂O₃ + 3CO → 2Fe + 3CO₂     ΔH = +20,5 kJ

Jadi, perubahan entalpi (ΔH) pada reaksi target sebesar ΔH = +20,5 kJ

_________________
Catatan:
⇒ Fe₂O₃ langsung turun ke bawah pada reaksi target
⇒ ¹/₃CO ditambahkan dengan 2CO dan ²/₃CO menjadi 3CO pada reaksi target
⇒ 2Fe langsung turun ke bawah pada reaksi target
⇒ ¹/₃CO₂ ditambahkan dengan 2CO₂ dan ²/₃CO₂ menjadi 3CO₂ pada reaksi target
⇒ Berdasarkan posisinya yang saling bersilangan, 2FeO pada reaksi-2 dan reaksi-3 saling menghabisi (mirip dengan saling mengurangi), hal yang sama terjadi antara ²/₃Fe₃O₄ pada reaksi-1 dengan ²/₃Fe₃O₄ pada reaksi-3
