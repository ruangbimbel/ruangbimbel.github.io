Terbukti bahwa [tex]\boldsymbol{(1+cot^2x)(1-sin^2x)=cot^2x}[/tex].

PEMBAHASAN

Trigonometri merupakan ilmu matematika yang mempelajari hubungan antara panjang dan sudut pada segitiga. Dalam trigonometri terdapat banyak rumus identitas, antara lain :

[tex]sin^2x+cos^2x=1[/tex]

[tex]cotx=\frac{1}{tanx}=\frac{cosx}{sinx}[/tex]

.

DIKETAHUI

[tex](1+cot^2x)(1-sin^2x)=cot^2x[/tex]

.

DITANYA

Buktikan identitas trigonometri di atas.

.

PENYELESAIAN

[tex](1+cot^2x)(1-sin^2x)=\left ( 1+\frac{cos^2x}{sin^2x} \right )(cos^2x)[/tex]

[tex](1+cot^2x)(1-sin^2x)=\left ( \frac{sin^2x+cos^2x}{sin^2x} \right )(cos^2x)[/tex]

[tex](1+cot^2x)(1-sin^2x)=\left ( \frac{1}{sin^2x} \right )(cos^2x)[/tex]

[tex](1+cot^2x)(1-sin^2x)=\left ( \frac{cosx}{sinx} \right )^2[/tex]

[tex](1+cot^2x)(1-sin^2x)=cot^2x~~(terbukti)[/tex]

.

KESIMPULAN

Terbukti bahwa [tex]\boldsymbol{(1+cot^2x)(1-sin^2x)=cot^2x}[/tex].

28. buktikan bahwa tan x-cot x / tan x +cot x=1-2 cos 2 x​


Pembuktian bahwa [tex] \frac{tan \: x - cot \: x}{tan \: x + cot \: x} = 1 - 2 {cos}^{2} x[/tex].

Identitas trigonometri adalah persamaan - persamaan yang dibuat untuk menyatakan hubungan antara dua atau lebih perbandingan trigonometri dalam operasi hitung guna memudahkan kita untuk menyelesaikan persoalan - persoalan yang menyangkut dua atau lebih perbandingan trigonometri.

Identitas trigonometri terdiri dari dua ruas, kanan dan kiri, di mana salah satu ruas akan dijabarkan dan dicocokkan dengan ruas lainnya sehingga menghasilkan pernyataan yang membuktikan bahwa kedua ruas tersebut berhubungan.

Agar lebih jelasnya, simak pembahasan soal berikut.

PEMBAHASAN :

Perhatikan kembali soal di atas.

Kita diminta untuk membuktikan bahwa [tex] \frac{tan \: x - cot \: x}{tan \: x + cot \: x} = 1 - 2 {cos}^{2} x[/tex]. Maka kita harus memilih salah satu ruas untuk diselesaikan sehingga memiliki hasil yang sama dengan ruas lain. Saya akan pilih ruas kiri.

[tex] \frac{tan \: x - cot \: x}{tan \: x + cot \: x} = ( \frac{sin \: x}{cos \: x} - \frac{cos \: x}{sin \: x} ) \div ( \frac{sin \: x}{cos \: x} + \frac{cos \: x}{sin \: x} ) \\ = ( \frac{ {sin}^{2}x - {cos}^{2}x }{sin \: x.cos \: x} ) \div( \frac{ {sin}^{2}x + {cos}^{2}x }{sin \: x.cos \: x} ) \\ =( \frac{ {sin}^{2}x - {cos}^{2}x }{sin \: x.cos \: x} ) \times ( \frac{sin \: x.cos \: x}{ {sin}^{2}x + {cos}^{2}x } ) \\ = ( \frac{ {sin}^{2}x - {cos}^{2} x }{ {sin}^{2}x + {cos}^{2}x } ) \\ =\frac{ {sin}^{2}x - {cos}^{2}x }{1} [/tex]

= sin²x - cos²x

= (sin²x + cos²x) - cos²x - cos²x

= 1 - 2cos²x

Jadi, TERBUKTI bahwa [tex] \frac{tan \: x - cot \: x}{tan \: x + cot \: x} = 1 - 2 {cos}^{2} x[/tex].
