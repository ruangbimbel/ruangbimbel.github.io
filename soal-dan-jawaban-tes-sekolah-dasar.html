Jawaban:

57

Penjelasan dengan langkah-langkah:

Maka :

- soal dijawab benar = 34

- soal kosong = 5

- soal salah

= 50 - (34+5)

= 50 - 39

= 11

Jadi Nilai Raka :

= (34×2) + (11×(-1) ) + (5×0)

= 68 + (-11) +0

= 57
