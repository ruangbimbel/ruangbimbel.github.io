9. Biasanya terdapat beberapa tahapan tes untuk masuk perguruan tinggi kedinasan. yaitu Seleksi Administrasi, tes kemampuan akademik dan tes kemampuan dasar dan tes wawancara. Karena banyaknya peminat, maka peluang untuk diterimapun sangat terbatas. Apabila terdapat 20.000 peminat masuk salah satu perguruan tinggi kedinasan dengan peluang lolos masing-masing tahapan berturut-turut 0,9 ; 0,15 ; 0,3 dan 0,8, maka banyak yang memenuhi syarat lulus adalah … orang​


Jawab:

Setelah diadakannya empat seleksi pada tahapan tes untuk masuk perguruan tinggi tersebut, maka banyak yang memenuhi syarat lulus adalah 648 orang.

Penjelasan dengan langkah-langkah:

Dari soal tersebut dapat diketahui bahwa dari 20.000 peminat masuk salah satu perguruan tinggi, rasio untuk lolos seleksi administrasi sebanyak 0,9 orang, rasio lolos tes kemampuan akademik sebanyak 0,15 orang, lalu lolos tes kemampuan dasar sebanyak 0,3 orang, dan yang terakhir lolos tes wawancara sebanyak 0,8 orang.

lolos seleksi administrasi: 0,9 x 20.000 = 18.000lolos tes kemampuan akademik; 0,15 x 18.000= 2.700lolos tes kemampuan dasar: 0,3 x 2.700 = 810lolos tes wawancara: 0,8 x 810 = 648

(6). Sin x + 1/2(√3) = 0, untuk 0 ≤ x ≤ 2π
sin x = - 1/2(√3)
sin x = 4π/3

- Mencari x dengan α + k(360)
Untuk k = 0
Sin x = 4π/3 + 0
Sin x = 4π/3
Untuk k = 1
Sin x = 4π/3 + 2π
Sin x = 10π/3 (T.M)

- Mencari x dengan (180 - α) + k(360)
Untuk k = 0
x = π - 4π/3
x = - π/3 (T. M)
Untuk k = 1
x = (π - 4π/3) + 2π
x = 5π/3
HP = {4π/3, 5π/3}

(7). Cos x = (1/2)√2 untuk 0 ≤ x ≤ 360°

Mencari x dengan α + k(360)
- Untuk k = 0
x = 45°
- Untuk k = 1
x = 45° + 360
x = 405° (T.M)

Mencari x dengan - a + k(360)
Untuk k = 0
x = - 45° (T.M)
Untuk x = 1
x = - 45° + 360
x = 315°
HP = {45°, 315°}
