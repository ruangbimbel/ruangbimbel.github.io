Kelas : 11
Mapel : Matematika
Kategori : Trigonometri Lanjut, Limit
Kata Kunci : Sudut rangkap, persamaan trigonometri, Limit x mendekati tak terhingga
Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)

31. Untuk dapat diterima di suatu pendidikan, harus lulus tes matematika dengan nilai tak kurang dari 7, dan tes biologi dengan nilai tidak kurang dari 5, sedangkan jumlah nilai matematika dan biologi tidak boleh kurang dari 13. Seorang calon dengan jumlah dua kali nilai matematika dan 3 kali nilai biologinya sama dengan 30. Calon itu … A. pasti ditolak B. pasti diterima C. diterima asal nilai matematika tidak lebih dari 9 D. diterima asal nilai biologi tidak kurang dari 5 Contoh Soal Ujian Akhir Sekolah PAT


Jawaban:

calon tersebut pasti ditolak (A. pasti ditolak).

Penjelasan dengan langkah-langkah:

Untuk menjawab pertanyaan ini, kita perlu menguraikan persyaratan dan informasi yang diberikan.

Persyaratan:

- Nilai tes matematika minimal 7.

- Nilai tes biologi minimal 5.

- Jumlah nilai matematika dan biologi tidak kurang dari 13.

Informasi tambahan:

- Jumlah dua kali nilai matematika dan tiga kali nilai biologi sama dengan 30.

Kita dapat menyelesaikan masalah ini dengan menggunakan pendekatan aljabar. Mari kita tentukan nilai matematika sebagai "x" dan nilai biologi sebagai "y".

Persamaan-persamaan yang kita dapatkan dari informasi yang diberikan adalah:

1. x ≥ 7

2. y ≥ 5

3. x + y ≥ 13

4. 2x + 3y = 30

Dari persamaan 4, kita dapat mengubahnya menjadi:

2x = 30 - 3y

x = (30 - 3y) / 2

Kita dapat menggunakan substitusi untuk menggantikan x dalam persamaan 3:

(30 - 3y) / 2 + y ≥ 13

30 - 3y + 2y ≥ 26

-y ≥ -4

y ≤ 4

Dari persamaan 2, kita tahu bahwa y harus lebih besar dari atau sama dengan 5, tetapi dari hasil di atas, y juga harus lebih kecil dari atau sama dengan 4. Oleh karena itu, tidak ada nilai y yang memenuhi persyaratan.

Jadi, berdasarkan informasi yang diberikan, calon tersebut pasti ditolak (A. pasti ditolak).

Seorang pendaftar nilai matematikanya jika dikalikan 2 lalu ditambah tiga kali nilai biologinya sma dgn 300. Apakah pendaftar tersebut akan diterima? Berikan alesanmu

pendaftar tersebut tidak diterima karena nilai matematika kurang dari 70 dan jumlah nilai kurang dari 130.

syarat pendaftar diterima:

nilai matematika ≤ 70

nilai biologi ≤ 50

jumlah nilai ≤ 130

misalkan:

x = nilai matematika

y = nilai biologi

nilai peserta = 2x + 3y = 300

cari nilai x

2x = 300 – 3y

masukkan 2x = 300 – 3y ke persamaan 2x + 3y = 300

2x + 3y = 300

2(300 – 3y) + 3y = 300

600 – 6y + 3y = 300

– 3y = 300 – 600

– 3y = –300

y = –300 : (–3)

y = 100

masukkan y = 100 ke persamaan 2x + 3y = 300

2x + 3y = 300

2x + 3(100) = 300

2x + 300 = 300

2x = 300 – 300

2x = 0

x = 0

hitung jumlah nilai

x + y = 0 + 100 = 100

jadi nilai pendaftar:

matematika = 0 (kurang dari 70)
