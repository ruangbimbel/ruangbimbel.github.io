21. Lili and Bibi loves to play a game called tic tac toe. They have played N games in a row. For each game, it is known who was the winner, Lili or Bibi. Now Lili wonders, who won more games, she or Bibi? Help her determine this by making a program that output who wins more. Format Input The first line of input consist integer T, the number of the testcase. The next X lines consist integer N and string S, consisting of N uppercase letters 'B' when Bibi won the game, 'L' when Lili won the game and 'T' when it is tie. Format Output Output "Lili" (without quotes) if Lili won more than Bibi, "Bibi" (without quotes) if Bibi won more than Lili, "None" (without quotes) if they won the same number of games. Constraints 1 <= T <= 100 1 <= N <= 100


Jawaban:

Berikut adalah code dalam bahasa C++

#include <iostream>

#include <string>

using namespace std;

int main(){

int t,n;

string s;

cin>>t;

while(t- -){

int lili=0,bibi=0;

cin>>n>>s;

for(int i = 0; i<n; i++){

if( s[i] =='L' ) lili++;

else if( s[i] == 'B' ) bibi++;

}

if(lili > bibi ) cout<<"Lili\n";

else if(lili == bibi) cout<<"None\n";

else cout<<"Bibi\n";

}

return 0;

}

Penjelasan:

pertama, siapkan variabel untuk masing2 input. variabel t untuk jumlah testcase, n untuk panjang string dan s untuk string yang dimaksud.

lalu, siapkan juga variabel untuk menampung berapa kali setiap orang menang dan diberi inisiasi awal dengan nilai 0, misal variabel bibi ketika bibi yang menang dan variabel lili ketika lili yang menang. kita tidak menyiapkan variabel tie karena hal ini tidak berdampak pada apapun.

iterasi setiap karakter pada string s menggunakan looping. jika s[i] sama dengan "B", maka variabel bibi akan mengalami increment dengan 1. jika s[i] sama dengan " L", maka variabel lili akan mengalami increment dengan 1. tetapi jika s[i] sama dengan "T", maka tidak akan berpengaruh apa apa.

setelah semua karakter pada string s telah diperiksa, maka lakukan pengecekan pada variabel bibi dan lili. jika bibi lebih besar dari lili, maka output akan menampilkan "Bibi" tanpa tanda petik. jika bibi sama dengan lili , maka output akan menampilkan "None" tanpa tanda petik. dan terakhir jika bibi lebih kecil daripada lili , maka output akan menampilkan "Lili" tanpa tanda petik.

jangan lupa untuk selalu memberi newline pada akhir dari setiap baris output
