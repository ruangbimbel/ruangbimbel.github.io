23. tentukan himpunan penyelesaian dari persamaan berikut 1. 2x² - 18x = 0 2. X² - 6x -27 = 0 3. 2x² - x - 15=0


Jawaban:

Untuk menentukan himpunan penyelesaian dari setiap persamaan quadratik, kita dapat menggunakan rumus diskriminan. Diskriminan didefinisikan sebagai \( D = b^{2} - 4ac \), di mana a, b, dan c adalah koefisien persamaan quadratik dalam bentuk umum \( ax^{2} + bx + c = 0 \). Berikut adalah langkah-langkah untuk menyelesaikan setiap persamaan:

1. Persamaan \( 2x^{2} - 18x = 0 \):

Koefisien a = 2, b = -18, dan c = 0.

Diskriminan D = \((-18)^{2} - 4(2)(0) = 324\).

Karena diskriminan positif (D > 0), maka persamaan ini memiliki dua akar real.

Himpunan penyelesaiannya adalah semua bilangan real.

2. Persamaan \( x^{2} - 6x - 27 = 0 \):

Koefisien a = 1, b = -6, dan c = -27.

Diskriminan D = \((-6)^{2} - 4(1)(-27) = 144\).

Karena diskriminan positif (D > 0), maka persamaan ini memiliki dua akar real.

Himpunan penyelesaiannya adalah semua bilangan real.

3. Persamaan \( 2x^{2} - x - 15 = 0 \):

Koefisien a = 2, b = -1, dan c = -15.

Diskriminan D = \((-1)^{2} - 4(2)(-15) = 1 + 120 = 121\).

Karena diskriminan positif (D > 0), maka persamaan ini memiliki dua akar real.

Himpunan penyelesaiannya adalah semua bilangan real.

Jadi, himpunan solusi dari setiap persamaan adalah semua bilangan real
