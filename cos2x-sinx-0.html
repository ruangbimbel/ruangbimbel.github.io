27. Lim phi/2 (Cos2x + Sinx) / (1-sinx)​


Jawaban:

jangan lupa follow dan jadikan jawaban tercerdas

Penjelasan dengan langkah-langkah:

Untuk menyederhanakan ekspresi trigonometri ini, kita akan menggunakan identitas trigonometri dasar. Pertama, mari kita menulis ulang ekspresi tersebut dengan menggunakan identitas trigonometri:

Lim x → π/2 [(Cos^2x + Sinx) / (1 - Sinx)]

Langkah 1: Menerapkan identitas trigonometri Cos^2x = 1 - Sin^2x

Lim x → π/2 [(1 - Sin^2x + Sinx) / (1 - Sinx)]

Langkah 2: Menggabungkan suku dengan Sinx

Lim x → π/2 [(1 + Sinx - Sin^2x) / (1 - Sinx)]

Langkah 3: Menerapkan identitas trigonometri (1 - Sinx) = Cos^2x

Lim x → π/2 [(1 + Sinx - Sin^2x) / Cos^2x]

Langkah 4: Menerapkan identitas trigonometri Cos^2x = 1 - Sin^2x

Lim x → π/2 [(1 + Sinx - (1 - Sin^2x)) / (1 - Sin^2x)]

Lim x → π/2 [(1 + Sinx - 1 + Sin^2x) / (1 - Sin^2x)]

Lim x → π/2 [(Sin^2x + Sinx) / (1 - Sin^2x)]

Langkah 5: Menerapkan identitas trigonometri Sin^2x = 1 - Cos^2x

Lim x → π/2 [(1 - Cos^2x + Sinx) / (1 - (1 - Cos^2x))]

Lim x → π/2 [(1 - Cos^2x + Sinx) / Cos^2x]

Langkah 6: Menerapkan identitas trigonometri Cos^2x = 1 - Sin^2x

Lim x → π/2 [(1 - (1 - Sin^2x) + Sinx) / (1 - Sin^2x)]

Lim x → π/2 [(1 - 1 + Sin^2x + Sinx) / (1 - Sin^2x)]

Lim x → π/2 [(Sin^2x + Sinx) / (1 - Sin^2x)]

Langkah 7: Menggunakan identitas trigonometri Sin^2x + Sinx = Sinx(Sinx + 1)

Lim x → π/2 [(Sinx(Sinx + 1)) / (1 - Sin^2x)]

Langkah 8: Menerapkan identitas trigonometri 1 - Sin^2x = Cos^2x

Lim x → π/2 [(Sinx(Sinx + 1)) / Cos^2x]

Langkah 9: Menggabungkan hasil

Lim x → π/2 [(Sinx(Sinx + 1)) / (1 - Sinx)(1 + Sinx)]

Lim x → π/2 [Sinx / (1 - Sinx)]

Langkah 10: Menerapkan nilai batas

Lim x → π/2 [1 / (-1)]

Lim x → π/2 [-1]

Jadi, hasil dari lim x → π/2 [(Cos^2x + Sinx) / (1 - Sinx)] adalah -1.
