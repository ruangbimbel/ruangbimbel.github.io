26. f(x)= cos 3x : sin 3x , f'(x)=

v = sin(3x) → v' = 3cos(3x)

= (-3sin(3x) sin(3x) - 3 cos(3x) cos(3x))/sin²(3x))
= (-3 sin²(3x) - 3 cos²(3x))/sin²(3x)
= -3(sin²(3x) + cos²(3x))/sin²(3x)
= -3(1)/sin²(3x)
= -3 csc²(3x)

#sejutapohon

Jika

[tex]f(x) = (3 {x}^{2} + 3x)(x + 5)[/tex]

Maka, f(0)+f(1) = 36

Pembahasan

Agar mudah, kita uraikan fungsinya terlebih dahulu,

[tex]f(x) = (3 {x}^{2} + 3x)(x + 5) \\ f(x) = 3 {x}^{3} + 15 {x}^{2} + 3 {x}^{2} + 15x \\ f(x) = 3 {x}^{3} + 18 {x}^{2} + 15x[/tex]

Masukkan x = 0 dan x = 1,

[tex]f(x) = 3 {x}^{3} + 18 {x}^{2} + 15x \\ f(0) = 0 + 0 + 0 = 0[/tex]

[tex]f(x) = 3 {x}^{3} + 18 {x}^{2} + 15x \\ f(1) = 3 \times 1 + 18 \times 1 + 15 \times 1 \\ f(1) = 36[/tex]

Maka,

f(0) + f(1) = 0 + 36 = 36
