5. Persamaan kuadrat yang akar akarnya 7 dan -4 adalah A. X²-11x-28=0 B. x²+3x+28=0 C. x²+3x-28=0 D. x²-3x-28=0


Jawaban:

Untuk menentukan persamaan kuadrat yang memiliki akar-akar tertentu, kita dapat menggunakan persamaan kuadrat dengan rumus (x - r1)(x - r2) = 0, di mana r1 dan r2 adalah akar-akar persamaan.

Dalam kasus ini, kita memiliki akar 7 dan -4. Maka kita dapat menggunakan rumus yang dijelaskan di atas untuk mencari persamaan kuadratnya.

(x - 7)(x + 4) = 0

Mengalikan persamaan ini akan menghasilkan:

x² - 7x + 4x - 28 = 0

x² - 3x - 28 = 0

Jadi, persamaan kuadrat yang memiliki akar-akar 7 dan -4 adalah D. x² - 3x - 28 = 0.

1.

x = 7 | x = -4

(x - 7)(x + 4)

= x^2 + 4x - 7x - 28

= x^2 - 3x - 28 (D.)

Diketahui persamaan kuadrat : x² + 3x - 28 = 0, maka nilai a, b dan c berturut-turut adalah 1, 3 dan - 28

Pembahasan

Diketahui

Persamaan kuadrat : x² + 3x - 28 = 0

Ditanya

Nilai a, b dan c

Solusi

Perlu diketahui bentuk umum persamaan kuadrat adalah ax² + bx + c = 0, dimana a adalah koefisien x², b koefisien x dan c adalah konstanta. Dari sini hanya perlu mengubah bentuk persamaan kuadrat yang diketahui ke bentuk umum persamaan kuadrat.

Maka:

x² + 3x - 28 = 0

1x² + 3x + ( - 28 )  = 0

didapat:

a = 1b = 3c = 28

∴ Jadi, nilai a, b, dan c berturut-turut adalah 1, - 3 dan 10
