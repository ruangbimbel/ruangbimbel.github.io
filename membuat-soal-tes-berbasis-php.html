34. Apa yang sebaiknya kita lakukan setelah mendesain logo menggunakan corelDraw, agar dapat di tampilkan dalam halaman web berbasis PHP


Hal terbaik yang bisa kita lakukan setelah mendesain logo dalam Corel Draw adalah dengan cara mengekspor logo tersebut kedalam bentuk file gambar yang mendukung format website dengan tujuan agar bisa ditampilkan  dalam apliasi web berbasis PHP.

Pembahasan:

Pada dasarnya menampilkan sebuah objek gambar pada sebuah web akan sama, baik untuk yang berbasis PHP maupun yang lainnya selama format gambar tersebut memang mendukung untuk ditampilkan oleh peramban atau browser.

Format apa saja yang bisa ditampilkan tersebut? Ada 2 jenis format gambar yang bisa kita tampillkan dallam website yaitu bitmap, seperti jpg, png, gif atau dalam bentuk SVG (scalable vector graphic). Soal mana yang terbaik akan tergantung kebutuhan yang ada, karena masing-masing memiliki keunggulan tersendiri.

Sebagai panduan berikut ini adalah langkah-lankah untuk mengekspor logo maupun vektor di corel draw ke dalam format gambar yang mendukung aplikasi web.

Seleksi objek/logo yang akan dieksporKarena logo akan dipruntukan untuk web maka kita akan menggunakan fikur export for web, caranya klik menu file >> export for >> webPilih output filenya, jika ingin berlatar belakang tranparan maka pilih pngKLik save as lalu simpan di direktori yang kamu kehendakiBuka direktori dimana logo tadi tersimpan lalu pindahkan ke direktori aset statis aplikasi web mu
